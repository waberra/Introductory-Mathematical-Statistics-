üìä 3.1 Continuous Random Variables

Definition: Continuous Random Variable

A random variable $X$ is continuous if there exists a non-negative function $f(x)$ called the probability density function (PDF) such that for any interval $[a,b]$:

$P(a \leq X \leq b) = \int_a^b f(x) \, dx$

Key Properties of Continuous Random Variables

  • Point Probability: $P(X = c) = 0$ for any constant $c$
  • Interval Equivalence: $P(a < X < b) = P(a \leq X \leq b) = P(a < X \leq b) = P(a \leq X < b)$
  • Range: Uncountable (typically intervals of real numbers)

Example 3.1: Height Measurements

Experiment: Measure the height of a randomly selected adult

Random Variable: $X$ = height in inches

Range: $(0, \infty)$ or realistically $[48, 84]$ inches

Key Insight: $P(X = 70.0000...)$ = 0, but $P(69.5 \leq X \leq 70.5) > 0$

Discrete vs. Continuous Comparison

Property Discrete Continuous
Range Countable set Uncountable set (intervals)
Probabilities PMF: $p(x) = P(X = x)$ PDF: $f(x)$ (not a probability)
Point Probability $P(X = c) \geq 0$ $P(X = c) = 0$
Intervals $P(a \leq X \leq b) = \sum_{x=a}^b p(x)$ $P(a \leq X \leq b) = \int_a^b f(x) dx$

üìà 3.2 Probability Density Function & Cumulative Distribution Function

Probability Density Function (PDF)

For a continuous random variable $X$, the PDF $f(x)$ satisfies:

$P(a \leq X \leq b) = \int_a^b f(x) \, dx$

Properties:

  • $f(x) \geq 0$ for all $x$
  • $\int_{-\infty}^{\infty} f(x) \, dx = 1$
  • $f(x)$ is not a probability (can be > 1)

Cumulative Distribution Function (CDF)

The CDF $F(x)$ for continuous variables is:

$F(x) = P(X \leq x) = \int_{-\infty}^{x} f(t) \, dt$

Relationship with PDF:

$f(x) = \frac{d}{dx} F(x) = F'(x)$

Example 3.2: Simple Uniform Distribution

Let $X$ be uniform on $[0, 2]$ with PDF:

$f(x) = \begin{cases} \frac{1}{2} & \text{if } 0 \leq x \leq 2 \\ 0 & \text{otherwise} \end{cases}$

CDF: $F(x) = \begin{cases} 0 & \text{if } x < 0 \\ \frac{x}{2} & \text{if } 0 \leq x \leq 2 \\ 1 & \text{if } x > 2 \end{cases}$

Example: $P(0.5 \leq X \leq 1.5) = \int_{0.5}^{1.5} \frac{1}{2} dx = \frac{1}{2} \cdot 1 = 0.5$

üéØ 3.3 Expected Value and Variance

Expected Value for Continuous Random Variables

For a continuous random variable $X$ with PDF $f(x)$:

$E[X] = \mu = \int_{-\infty}^{\infty} x \cdot f(x) \, dx$

Function of X: $E[g(X)] = \int_{-\infty}^{\infty} g(x) \cdot f(x) \, dx$

Variance for Continuous Random Variables

$\text{Var}(X) = \sigma^2 = E[(X - \mu)^2] = \int_{-\infty}^{\infty} (x - \mu)^2 f(x) \, dx$

Computational formula: $\text{Var}(X) = E[X^2] - (E[X])^2$

Example 3.3: Expected Value of Uniform[0,2]

For $X \sim \text{Uniform}(0, 2)$ with $f(x) = \frac{1}{2}$ on $[0,2]$:

$E[X] = \int_0^2 x \cdot \frac{1}{2} \, dx = \frac{1}{2} \int_0^2 x \, dx = \frac{1}{2} \cdot \frac{x^2}{2}\Big|_0^2 = \frac{1}{4} \cdot 4 = 1$

Intuition: The expected value is the center of the uniform interval.

üìè 3.4 Uniform Distribution

Uniform Distribution: $X \sim \text{Uniform}(a, b)$

The uniform distribution represents equal probability density over an interval [a,b].

$f(x) = \begin{cases} \frac{1}{b-a} & \text{if } a \leq x \leq b \\ 0 & \text{otherwise} \end{cases}$

Uniform Distribution Properties

  • Mean: $E[X] = \frac{a+b}{2}$ (midpoint of interval)
  • Variance: $\text{Var}(X) = \frac{(b-a)^2}{12}$
  • Standard Deviation: $\sigma = \frac{b-a}{2\sqrt{3}}$
  • CDF: $F(x) = \frac{x-a}{b-a}$ for $a \leq x \leq b$

Example 3.4: Random Number Generator

Problem: A computer generates random numbers uniformly on [0,1]. What's P(X > 0.7)?

Solution:

  • X ~ Uniform(0,1), so f(x) = 1 for 0 ‚â§ x ‚â§ 1
  • P(X > 0.7) = ‚à´‚ÇÄ.‚Çá¬π 1 dx = 1 - 0.7 = 0.3
  • Alternatively: P(X > 0.7) = (1-0.7)/(1-0) = 0.3

Key Characteristics

Shape: Rectangular (flat) PDF - constant density

Symmetry: Symmetric around the mean Œº = (a+b)/2

Maximum entropy: Among all distributions on [a,b], uniform has maximum entropy

Memoryless property: No memoryless property (unlike exponential)

Example 3.5: Bus Arrival Times

Problem: Buses arrive every 10 minutes. If you arrive at a random time, your waiting time is Uniform(0,10). Find the probability you wait more than 7 minutes.

Solution:

  • X ~ Uniform(0,10), so f(x) = 1/10 for 0 ‚â§ x ‚â§ 10
  • P(X > 7) = ‚à´‚Çá¬π‚Å∞ (1/10) dx = (1/10)(10-7) = 3/10 = 0.3
  • Expected waiting time: E[X] = (0+10)/2 = 5 minutes

Quantiles and Percentiles

For X ~ Uniform(a,b), the p-th percentile is:

$x_p = a + p(b-a)$ where $0 \leq p \leq 1$

Examples: Median = (a+b)/2, 25th percentile = a + 0.25(b-a)

Example 3.6: Quality Control

Problem: A machine produces parts with diameters uniformly distributed between 9.8 and 10.2 mm. What fraction meet specifications of 9.9 to 10.1 mm?

Solution:

  • X ~ Uniform(9.8, 10.2), so b-a = 0.4 mm
  • P(9.9 ‚â§ X ‚â§ 10.1) = (10.1-9.9)/(10.2-9.8) = 0.2/0.4 = 0.5
  • 50% of parts meet specifications

‚ö° 3.5 Exponential Distribution

Exponential Distribution: $X \sim \text{Exponential}(\lambda)$

The exponential distribution models waiting times between events in a Poisson process.

$f(x) = \lambda e^{-\lambda x}$ for $x \geq 0$, $\lambda > 0$

Parameter: Œª is the rate parameter (events per unit time)

Exponential Distribution Properties

  • Mean: $E[X] = \frac{1}{\lambda}$ (average waiting time)
  • Variance: $\text{Var}(X) = \frac{1}{\lambda^2}$
  • Standard Deviation: $\sigma = \frac{1}{\lambda}$
  • CDF: $F(x) = 1 - e^{-\lambda x}$ for $x \geq 0$
  • Memoryless Property: $P(X > s + t | X > s) = P(X > t)$

Example 3.7: Customer Service Calls

Problem: Calls arrive at rate Œª = 2 per hour. What's the probability the next call takes more than 30 minutes?

Solution:

  • X ~ Exponential(2), where X is waiting time in hours
  • P(X > 0.5) = 1 - F(0.5) = 1 - (1 - e^(-2√ó0.5)) = e^(-1) ‚âà 0.368
  • About 36.8% chance of waiting more than 30 minutes

Memoryless Property

The exponential distribution is the only continuous distribution with the memoryless property:

$P(X > s + t | X > s) = P(X > t)$

Interpretation: Given you've waited s time units, the probability of waiting t more is the same as if starting fresh.

Example 3.8: Electronic Component Lifetimes

Problem: A component's lifetime follows Exponential(0.1) where time is in years. If it's survived 5 years, what's P(survives 3 more years)?

Solution:

  • By memoryless property: P(X > 8 | X > 5) = P(X > 3)
  • P(X > 3) = 1 - F(3) = e^(-0.1√ó3) = e^(-0.3) ‚âà 0.741
  • About 74.1% chance of surviving 3 more years

Relationship to Poisson Process

If events occur according to Poisson(Œªt) in time interval [0,t], then:

  • Waiting time until first event ~ Exponential(Œª)
  • Time between consecutive events ~ Exponential(Œª)
  • Examples: Radioactive decay, phone calls, equipment failures

Example 3.9: Website Traffic

Problem: Visitors arrive at rate 5 per minute. Find the median time until the next visitor.

Solution:

  • X ~ Exponential(5), where X is in minutes
  • For median, solve F(m) = 0.5: 1 - e^(-5m) = 0.5
  • e^(-5m) = 0.5, so -5m = ln(0.5) = -ln(2)
  • m = ln(2)/5 ‚âà 0.139 minutes ‚âà 8.3 seconds

The normal (Gaussian) distribution is the most important continuous distribution, modeling many natural phenomena.

Random Variable Transformation

Given a random variable X with known distribution, find the distribution of Y = g(X) where g is some function.

Key question: If we know f_X(x), what is f_Y(y)?

Transformation Technique (Monotonic Functions)

If Y = g(X) where g is strictly monotonic (increasing or decreasing) and differentiable:

$f_Y(y) = f_X(g^{-1}(y)) \left| \frac{d}{dy} g^{-1}(y) \right|$

Steps: (1) Find inverse g‚Åª¬π, (2) Find derivative, (3) Apply formula

For any transformation Y = g(X):

  1. Find CDF: F_Y(y) = P(Y ‚â§ y) = P(g(X) ‚â§ y)
  2. Express in terms of X: Find equivalent condition on X

  4. Differentiate: f_Y(y) = F'_Y(y)
