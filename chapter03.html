üìä 3.1 Continuous Random Variables

Definition: Continuous Random Variable

A random variable $X$ is continuous if there exists a non-negative function $f(x)$ called the probability density function (PDF) such that for any interval $[a,b]$:

$P(a \leq X \leq b) = \int_a^b f(x) \, dx$

Key Properties of Continuous Random Variables

  • Point Probability: $P(X = c) = 0$ for any constant $c$
  • Interval Equivalence: $P(a < X < b) = P(a \leq X \leq b) = P(a < X \leq b) = P(a \leq X < b)$
  • Range: Uncountable (typically intervals of real numbers)

Example 3.1: Height Measurements

Experiment: Measure the height of a randomly selected adult

Random Variable: $X$ = height in inches

Range: $(0, \infty)$ or realistically $[48, 84]$ inches

Key Insight: $P(X = 70.0000...)$ = 0, but $P(69.5 \leq X \leq 70.5) > 0$

Discrete vs. Continuous Comparison

Property Discrete Continuous
Range Countable set Uncountable set (intervals)
Probabilities PMF: $p(x) = P(X = x)$ PDF: $f(x)$ (not a probability)
Point Probability $P(X = c) \geq 0$ $P(X = c) = 0$
Intervals $P(a \leq X \leq b) = \sum_{x=a}^b p(x)$ $P(a \leq X \leq b) = \int_a^b f(x) dx$

üìà 3.2 Probability Density Function & Cumulative Distribution Function

Probability Density Function (PDF)

For a continuous random variable $X$, the PDF $f(x)$ satisfies:

$P(a \leq X \leq b) = \int_a^b f(x) \, dx$

Properties:

  • $f(x) \geq 0$ for all $x$
  • $\int_{-\infty}^{\infty} f(x) \, dx = 1$
  • $f(x)$ is not a probability (can be > 1)

Cumulative Distribution Function (CDF)

The CDF $F(x)$ for continuous variables is:

$F(x) = P(X \leq x) = \int_{-\infty}^{x} f(t) \, dt$

Relationship with PDF:

$f(x) = \frac{d}{dx} F(x) = F'(x)$

Example 3.2: Simple Uniform Distribution

Let $X$ be uniform on $[0, 2]$ with PDF:

$f(x) = \begin{cases} \frac{1}{2} & \text{if } 0 \leq x \leq 2 \\ 0 & \text{otherwise} \end{cases}$

CDF: $F(x) = \begin{cases} 0 & \text{if } x < 0 \\ \frac{x}{2} & \text{if } 0 \leq x \leq 2 \\ 1 & \text{if } x > 2 \end{cases}$

Example: $P(0.5 \leq X \leq 1.5) = \int_{0.5}^{1.5} \frac{1}{2} dx = \frac{1}{2} \cdot 1 = 0.5$

üéØ 3.3 Expected Value and Variance

Expected Value for Continuous Random Variables

For a continuous random variable $X$ with PDF $f(x)$:

$E[X] = \mu = \int_{-\infty}^{\infty} x \cdot f(x) \, dx$

Function of X: $E[g(X)] = \int_{-\infty}^{\infty} g(x) \cdot f(x) \, dx$

Variance for Continuous Random Variables

$\text{Var}(X) = \sigma^2 = E[(X - \mu)^2] = \int_{-\infty}^{\infty} (x - \mu)^2 f(x) \, dx$

Computational formula: $\text{Var}(X) = E[X^2] - (E[X])^2$

Example 3.3: Expected Value of Uniform[0,2]

For $X \sim \text{Uniform}(0, 2)$ with $f(x) = \frac{1}{2}$ on $[0,2]$:

$E[X] = \int_0^2 x \cdot \frac{1}{2} \, dx = \frac{1}{2} \int_0^2 x \, dx = \frac{1}{2} \cdot \frac{x^2}{2}\Big|_0^2 = \frac{1}{4} \cdot 4 = 1$

Intuition: The expected value is the center of the uniform interval.

üìè 3.4 Uniform Distribution

Uniform Distribution: $X \sim \text{Uniform}(a, b)$

$f(x) = \begin{cases} \frac{1}{b-a} & \text{if } a \leq x \leq b \\ 0 & \text{otherwise} \end{cases}$

Properties:

  • Mean: $E[X] = \frac{a+b}{2}$
  • Variance: $\text{Var}(X) = \frac{(b-a)^2}{12}$
  • CDF: $F(x) = \frac{x-a}{b-a}$ for $a \leq x \leq b$

‚ö° 3.5 Exponential Distribution

Exponential Distribution: $X \sim \text{Exponential}(\lambda)$

$f(x) = \lambda e^{-\lambda x}$ for $x \geq 0$, $\lambda > 0$

Properties:

  • Mean: $E[X] = \frac{1}{\lambda}$
  • Variance: $\text{Var}(X) = \frac{1}{\lambda^2}$
  • CDF: $F(x) = 1 - e^{-\lambda x}$ for $x \geq 0$
  • Memoryless Property: $P(X > s + t | X > s) = P(X > t)$

üîî 3.6 Normal Distribution

Normal Distribution: $X \sim N(\mu, \sigma^2)$

$f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$

Properties:

  • Mean: $E[X] = \mu$
  • Variance: $\text{Var}(X) = \sigma^2$
  • Symmetric: Bell-shaped curve centered at $\mu$
  • 68-95-99.7 Rule: Approximately 68%, 95%, 99.7% within 1, 2, 3 standard deviations

üîÑ 3.7 Transformations of Random Variables

Transformation Technique

If $X$ has PDF $f_X(x)$ and $Y = g(X)$ where $g$ is strictly monotonic, then:

$f_Y(y) = f_X(g^{-1}(y)) \left| \frac{d}{dy} g^{-1}(y) \right|$
