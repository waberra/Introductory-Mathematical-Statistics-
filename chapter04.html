üé≤ 4.1 Random Samples

Definition: Random Sample

Random variables $X_1, X_2, \ldots, X_n$ form a random sample of size $n$ if:

  1. Each $X_i$ has the same distribution as population $X$
  2. $X_1, X_2, \ldots, X_n$ are mutually independent

Notation: $X_1, X_2, \ldots, X_n \stackrel{\text{iid}}{\sim} X$

Example 4.1: Coin Flips

Population: Single coin flip with $P(\text{Heads}) = 0.5$

Random Sample: 10 independent flips

Joint PDF: $f(x_1, x_2, \ldots, x_{10}) = \prod_{i=1}^{10} f(x_i)$

üìä 4.2 Sample Statistics

Common Sample Statistics

  • Sample Mean: $\bar{X} = \frac{1}{n} \sum_{i=1}^n X_i$
  • Sample Variance: $S^2 = \frac{1}{n-1} \sum_{i=1}^n (X_i - \bar{X})^2$
  • Sample Standard Deviation: $S = \sqrt{S^2}$

Example 4.2: Sample Statistics

Sample: 2, 4, 6

Sample Mean: $\bar{X} = (2 + 4 + 6)/3 = 4$

Sample Variance: $S^2 = [(2-4)^2 + (4-4)^2 + (6-4)^2]/(3-1) = 4$

üìà 4.3 Distribution of Sample Mean

Sample Mean Properties

If $X_1, X_2, \ldots, X_n \stackrel{\text{iid}}{\sim} X$ with $E[X] = \mu$ and $\text{Var}(X) = \sigma^2$:

$E[\bar{X}] = \mu$ and $\text{Var}(\bar{X}) = \frac{\sigma^2}{n}$

Example 4.3: Standard Error

For $\sigma = 10$ and $n = 25$:

$SE(\bar{X}) = \frac{\sigma}{\sqrt{n}} = \frac{10}{\sqrt{25}} = \frac{10}{5} = 2$

üéØ 4.4 Central Limit Theorem

Central Limit Theorem (CLT)

For $X_1, X_2, \ldots, X_n \stackrel{\text{iid}}{\sim} X$ with $E[X] = \mu$ and $\text{Var}(X) = \sigma^2 < \infty$:

$\bar{X} \xrightarrow{d} N\left(\mu, \frac{\sigma^2}{n}\right)$ as $n \to \infty$

Key: Works for ANY population distribution!

CLT Applications

  • Universal: Population can be any shape
  • Sample size: Usually good for $n \geq 30$
  • Standardization: $Z = \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \sim N(0,1)$

üìä 4.5 Key Sampling Distributions

Chi-Square Distribution: $\chi^2_\nu$

If $Z_1, \ldots, Z_\nu \stackrel{\text{iid}}{\sim} N(0,1)$, then $Y = \sum Z_i^2 \sim \chi^2_\nu$

Properties: $E[Y] = \nu$, $\text{Var}(Y) = 2\nu$

t-Distribution: $t_\nu$

If $Z \sim N(0,1)$ and $Y \sim \chi^2_\nu$ independent, then $T = \frac{Z}{\sqrt{Y/\nu}} \sim t_\nu$

Properties: Symmetric, heavier tails than normal

F-Distribution: $F_{\nu_1,\nu_2}$

If $Y_1 \sim \chi^2_{\nu_1}$ and $Y_2 \sim \chi^2_{\nu_2}$ independent, then $F = \frac{Y_1/\nu_1}{Y_2/\nu_2} \sim F_{\nu_1,\nu_2}$

Used for: Comparing variances, ANOVA
