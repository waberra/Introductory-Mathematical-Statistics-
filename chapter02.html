üéØ 2.1 Random Variables

A random variable is a function that assigns a real number to each outcome of a random experiment.

Definition: Random Variable

Let $\Omega$ be a sample space. A random variable $X$ is a function $X: \Omega \rightarrow \mathbb{R}$ that assigns a real number $X(\omega)$ to each outcome $\omega \in \Omega$.

Example 2.1: Coin Tosses

Experiment: Toss a coin twice

Sample Space: $\Omega = \{HH, HT, TH, TT\}$

Random Variable: $X$ = number of heads

  • $X(HH) = 2$
  • $X(HT) = X(TH) = 1$
  • $X(TT) = 0$

Range of $X$: $\{0, 1, 2\}$

Types of Random Variables:

  • Discrete: Countable range (finite or countably infinite)
  • Continuous: Uncountable range (intervals of real numbers)

üìà 2.2 Probability Mass Function & Cumulative Distribution Function

Probability Mass Function (PMF)

For a discrete random variable $X$, the probability mass function (PMF) is:

$p(x) = P(X = x)$ for all $x \in \mathbb{R}$

Properties:

  • $p(x) \geq 0$ for all $x$
  • $p(x) = 0$ if $x$ is not in the range of $X$
  • $\sum_{x} p(x) = 1$ (sum over all possible values)

Cumulative Distribution Function (CDF)

The cumulative distribution function (CDF) is:

$F(x) = P(X \leq x) = \sum_{t \leq x} p(t)$

Properties:

  • $0 \leq F(x) \leq 1$ for all $x$
  • $F(x)$ is non-decreasing
  • $\lim_{x \to -\infty} F(x) = 0$ and $\lim_{x \to \infty} F(x) = 1$

Example 2.2: PMF and CDF for Coin Tosses

For $X$ = number of heads in 2 coin tosses:

$x$ $p(x) = P(X = x)$ $F(x) = P(X \leq x)$

üéØ 2.3 Expected Value

Expected Value (Mean)

The expected value of a discrete random variable $X$ is:

$E[X] = \mu_X = \sum_{x} x \cdot p(x)$

This is the weighted average of all possible values, where weights are the probabilities.

Properties of Expected Value

  • Linearity: $E[aX + b] = aE[X] + b$
  • Sum rule: $E[X + Y] = E[X] + E[Y]$
  • Function of X: $E[g(X)] = \sum_{x} g(x) \cdot p(x)$

Example 2.3: Expected Value of Dice Roll

For a fair six-sided die with $X \in \{1, 2, 3, 4, 5, 6\}$ and $p(x) = 1/6$ for each:

$E[X] = 1 \cdot \frac{1}{6} + 2 \cdot \frac{1}{6} + \cdots + 6 \cdot \frac{1}{6} = \frac{21}{6} = 3.5$

üìè 2.4 Variance and Standard Deviation

Definition of Variance

Variance measures the spread or dispersion of a random variable around its expected value.

$\text{Var}(X) = E[(X - \mu)^2] = E[X^2] - (E[X])^2$

where $\mu = E[X]$ is the mean.

Standard Deviation: $\sigma = \sqrt{\text{Var}(X)}$

Properties of Variance

  • Non-negative: $\text{Var}(X) \geq 0$
  • Constant: $\text{Var}(c) = 0$ for any constant $c$
  • Linear transformation: $\text{Var}(aX + b) = a^2\text{Var}(X)$
  • Independence: If $X$ and $Y$ independent, $\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y)$

Example 2.4: Computing Variance

Problem: Let X have PMF p(1) = 0.3, p(2) = 0.5, p(3) = 0.2. Find Var(X).

Solution:

  • $E[X] = 1(0.3) + 2(0.5) + 3(0.2) = 1.9$
  • $E[X^2] = 1^2(0.3) + 2^2(0.5) + 3^2(0.2) = 0.3 + 2.0 + 1.8 = 4.1$
  • $\text{Var}(X) = E[X^2] - (E[X])^2 = 4.1 - (1.9)^2 = 4.1 - 3.61 = 0.49$

Computational Methods

Method 1 - Definition: $\text{Var}(X) = E[(X-\mu)^2] = \sum (x-\mu)^2 p(x)$

Method 2 - Shortcut: $\text{Var}(X) = E[X^2] - (E[X])^2$

Advantage of Method 2: Often easier computationally, especially for known distributions.

Example 2.5: Variance of Indicator Variable

Problem: Let I be an indicator variable with P(I = 1) = p. Find Var(I).

Solution:

  • $E[I] = p$ (from previous section)
  • $E[I^2] = 1^2 \cdot p + 0^2 \cdot (1-p) = p$
  • $\text{Var}(I) = E[I^2] - (E[I])^2 = p - p^2 = p(1-p)$

Note: Variance is maximized when p = 1/2.

Standardization

The standardized variable is:

$Z = \frac{X - \mu}{\sigma} = \frac{X - E[X]}{\sqrt{\text{Var}(X)}}$

Properties: $E[Z] = 0$ and $\text{Var}(Z) = 1$

ü™ô 2.5 Binomial Distribution

Binomial Distribution

A random variable X follows a binomial distribution with parameters n and p if:

  • There are n independent trials
  • Each trial has probability p of success
  • X counts the number of successes

Notation: $X \sim \text{Binomial}(n, p)$ or $X \sim B(n, p)$

Binomial PMF

$P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}$ for $k = 0, 1, 2, \ldots, n$

where $\binom{n}{k} = \frac{n!}{k!(n-k)!}$ is the binomial coefficient.

Example 2.6: Coin Flipping

Problem: Flip a fair coin 5 times. What's P(exactly 3 heads)?

Solution:

  • X = number of heads, X ~ Binomial(5, 0.5)
  • $P(X = 3) = \binom{5}{3} (0.5)^3 (0.5)^2 = 10 \cdot (0.5)^5 = 10/32 = 5/16$

Binomial Parameters

Expected Value: $E[X] = np$

Variance: $\text{Var}(X) = np(1-p)$

Standard Deviation: $\sigma = \sqrt{np(1-p)}$

Example 2.7: Quality Control

Problem: A factory produces items with 2% defect rate. In a batch of 100, what's the expected number of defects and standard deviation?

Solution:

  • X ~ Binomial(100, 0.02)
  • $E[X] = np = 100 \times 0.02 = 2$
  • $\text{Var}(X) = np(1-p) = 100 \times 0.02 \times 0.98 = 1.96$
  • $\sigma = \sqrt{1.96} = 1.4$

Binomial Properties

  • Symmetry: When p = 0.5, distribution is symmetric around np
  • Skewness: Right-skewed when p < 0.5, left-skewed when p > 0.5
  • Normal approximation: When np ‚â• 5 and n(1-p) ‚â• 5, approximately normal
  • Sum property: If X ~ B(n‚ÇÅ,p) and Y ~ B(n‚ÇÇ,p), then X+Y ~ B(n‚ÇÅ+n‚ÇÇ,p)

üéØ 2.6 Geometric Distribution

Geometric Distribution

A random variable X follows a geometric distribution if:

  • Independent trials with constant success probability p
  • X = number of trials until first success (including the success)
  • Trials continue until success occurs

Notation: $X \sim \text{Geometric}(p)$ or $X \sim \text{Geom}(p)$

Geometric PMF

$P(X = k) = (1-p)^{k-1} p$ for $k = 1, 2, 3, \ldots$

Interpretation: (k-1) failures followed by 1 success

Example 2.8: Die Rolling

Problem: Roll a die until you get a 6. What's P(first 6 on trial 3)?

Solution:

  • X = trial number of first 6, X ~ Geometric(1/6)
  • $P(X = 3) = (1 - 1/6)^{3-1} \cdot (1/6) = (5/6)^2 \cdot (1/6) = 25/216$

Geometric Parameters

Expected Value: $E[X] = \frac{1}{p}$

Variance: $\text{Var}(X) = \frac{1-p}{p^2}$

Standard Deviation: $\sigma = \frac{\sqrt{1-p}}{p}$

Memoryless Property

The geometric distribution is memoryless:

$P(X > s + t | X > s) = P(X > t)$

Interpretation: Given no success in first s trials, the probability of waiting t more trials is the same as starting fresh.

Example 2.9: Expected Waiting Time

Problem: A basketball player makes 40% of free throws. On average, how many attempts until first success?

Solution:

  • X ~ Geometric(0.4)
  • $E[X] = 1/p = 1/0.4 = 2.5$ attempts

Alternative Formulation

Sometimes geometric is defined as number of failures before first success:

If Y = number of failures, then $P(Y = k) = (1-p)^k p$ for k = 0, 1, 2, ...

Relationship: If X = trials until success, then Y = X - 1

$E[Y] = \frac{1-p}{p}$, $\text{Var}(Y) = \frac{1-p}{p^2}$

Poisson Distribution

A random variable X follows a Poisson distribution if it counts the number of events occurring in a fixed time/space interval where:

  • Events occur at a constant average rate Œª
  • Events are independent
  • The probability of multiple events in a very small interval is negligible

Notation: $X \sim \text{Poisson}(\lambda)$ or $X \sim \text{Pois}(\lambda)$

Poisson PMF

$P(X = k) = \frac{e^{-\lambda} \lambda^k}{k!}$ for $k = 0, 1, 2, \ldots$

where Œª > 0 is the rate parameter.

Example 2.10: Customer Arrivals

Problem: A store receives an average of 3 customers per hour. What's P(exactly 2 customers in the next hour)?

Solution:

  • X = customers in next hour, X ~ Poisson(3)
  • $P(X = 2) = \frac{e^{-3} \cdot 3^2}{2!} = \frac{e^{-3} \cdot 9}{2} = \frac{9e^{-3}}{2} \approx 0.224$

Poisson Parameters

Expected Value: $E[X] = \lambda$

Variance: $\text{Var}(X) = \lambda$

Standard Deviation: $\sigma = \sqrt{\lambda}$

Special Property: Mean equals variance!

Poisson as Binomial Limit
