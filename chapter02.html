üéØ 2.1 Random Variables

A random variable is a function that assigns a real number to each outcome of a random experiment.

Definition: Random Variable

Let $\Omega$ be a sample space. A random variable $X$ is a function $X: \Omega \rightarrow \mathbb{R}$ that assigns a real number $X(\omega)$ to each outcome $\omega \in \Omega$.

Example 2.1: Coin Tosses

Experiment: Toss a coin twice

Sample Space: $\Omega = \{HH, HT, TH, TT\}$

Random Variable: $X$ = number of heads

  • $X(HH) = 2$
  • $X(HT) = X(TH) = 1$
  • $X(TT) = 0$

Range of $X$: $\{0, 1, 2\}$

Types of Random Variables:

  • Discrete: Countable range (finite or countably infinite)
  • Continuous: Uncountable range (intervals of real numbers)

üìà 2.2 Probability Mass Function & Cumulative Distribution Function

Probability Mass Function (PMF)

For a discrete random variable $X$, the probability mass function (PMF) is:

$p(x) = P(X = x)$ for all $x \in \mathbb{R}$

Properties:

  • $p(x) \geq 0$ for all $x$
  • $p(x) = 0$ if $x$ is not in the range of $X$
  • $\sum_{x} p(x) = 1$ (sum over all possible values)

Cumulative Distribution Function (CDF)

The cumulative distribution function (CDF) is:

$F(x) = P(X \leq x) = \sum_{t \leq x} p(t)$

Properties:

  • $0 \leq F(x) \leq 1$ for all $x$
  • $F(x)$ is non-decreasing
  • $\lim_{x \to -\infty} F(x) = 0$ and $\lim_{x \to \infty} F(x) = 1$

Example 2.2: PMF and CDF for Coin Tosses

For $X$ = number of heads in 2 coin tosses:

$x$ $p(x) = P(X = x)$ $F(x) = P(X \leq x)$

üéØ 2.3 Expected Value

Expected Value (Mean)

The expected value of a discrete random variable $X$ is:

$E[X] = \mu_X = \sum_{x} x \cdot p(x)$

This is the weighted average of all possible values, where weights are the probabilities.

Properties of Expected Value

  • Linearity: $E[aX + b] = aE[X] + b$
  • Sum rule: $E[X + Y] = E[X] + E[Y]$
  • Function of X: $E[g(X)] = \sum_{x} g(x) \cdot p(x)$

Example 2.3: Expected Value of Dice Roll

For a fair six-sided die with $X \in \{1, 2, 3, 4, 5, 6\}$ and $p(x) = 1/6$ for each:

$E[X] = 1 \cdot \frac{1}{6} + 2 \cdot \frac{1}{6} + \cdots + 6 \cdot \frac{1}{6} = \frac{21}{6} = 3.5$
